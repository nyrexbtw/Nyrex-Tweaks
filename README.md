# 🚀 Nyrex Tweaks v1.0

Nyrex Tweaks is a high-performance Windows optimization utility designed to reduce latency, debloat telemetry, and enhance the gaming experience. 

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Platform](https://img.shields.io/badge/platform-Windows-0078d7.svg)

## ✨ Features
* **Pill-Shaped UI:** Modern, dark-themed interface built with CustomTkinter.
* **162+ Tweaks:** Comprehensive optimization across 7 categories.
* **Safety First:** Integrated System Restore point creation before applying changes.
* **Admin-Ready:** Built-in manifest to ensure registry commands execute with full permissions.

## 🛠️ Installation
1. Download the latest `Nyrex Tweaks.exe` from the [Releases](https://github.com/YOUR_USERNAME/NyrexTweaks/releases) page.
2. Run the application (UAC prompt will appear for Admin rights).
3. Select a category and hit **APPLY**.
